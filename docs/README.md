# Map4D Service API document

Documentation website được tạo bằng [docsify](https://docsify.js.org/)

## Setup

```bash
npm i docsify-cli -g
```

## Preview

Goto docs folder,

```bash
docsify serve v2.0
```

Truy cập: http://localhost:3000

> OR

```bash
docsify serve .
```

Truy cập: http://localhost:3000/v1.0
